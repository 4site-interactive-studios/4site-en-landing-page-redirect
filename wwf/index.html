WWF Example Landing Page

This is a demonstration of the Homepage Takeover Redirect system

How This Works

This page demonstrates the landing page redirect functionality. The script will automatically redirect visitors to an Engaging Networks donation page on configured dates.

Note: For fastest performance, load this script in the <head> section as high up as possible. While Google Tag Manager (GTM) is an option, loading directly in the <head> is the most preferable method for optimal redirect speed. This example shows it inline for demonstration purposes.

Configuration & Testing

The redirect dates are configured in the urlsByDate object. Currently configured dates:

  • December 3, 2025 - Redirects to Engaging Networks page
  • December 30, 2025 - Redirects to Engaging Networks page
  • December 31, 2025 - Redirects to Engaging Networks page

Example Code

The following code is running on this page:

<script>
  // ============================================================================
  // CONFIGURATION
  // ============================================================================
  // Configure the dates and corresponding Engaging Networks page URLs
  // Date format: "YYYY-MM-DD" (e.g., "2025-12-03" for December 3rd, 2025)
  // Include any necessary URL parameters such as tracking codes in the URLs
  window.landingPageRedirectConfig = {
    urlsByDate: {
      "2025-12-03":
        "https://protect.worldwildlife.org/page/88974/donate/1?transaction.othamt1=TRACKINGCODE1",
      "2025-12-30":
        "https://protect.worldwildlife.org/page/88974/donate/1?transaction.othamt1=TRACKINGCODE2",
      "2025-12-31":
        "https://protect.worldwildlife.org/page/88974/donate/1?transaction.othamt1=TRACKINGCODE3",
    },
    suppressionCookie: "redirectSuppressed",
    suppressionDurationDays: 1,
  };
</script>
<script src="../redirect-script.js"></script>

Testing the Redirect

You can test the redirect functionality by adding the simulate-date parameter to the URL. For example:

  • index.html?simulate-date=2025-12-03 - Tests December 3rd, 2025 redirect
  • index.html?simulate-date=2025-12-30 - Tests December 30th, 2025 redirect
  • index.html?simulate-date=2025-12-31 - Tests December 31st, 2025 redirect
